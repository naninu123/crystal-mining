# Crystal Mining

> Crystal-themed mining simulation game

## Features

- Mine crystals
- Upgrade tools
- Market trading
- Guild system

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript
- Blockchain: Solidity, Rust, Web3.js
- AI: MiMo V2 Flash, Claude Code

## Installation

```bash
git clone https://github.com/naninu123/crystal-mining.git
cd crystal-mining
npm install
npm run dev
```
